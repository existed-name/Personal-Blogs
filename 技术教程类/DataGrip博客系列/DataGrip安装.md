# DataGrip安装

> 参考文章：[DataGrip安装文档](https://heuqqdmbyk.feishu.cn/wiki/FAa3wj0nYi4xGBksbFuchBK8nhe#STS5dq9hioTBPGxK6Cpc2NA1nff)
---

### 1、下载
1. 进入[官网](https://www.jetbrains.com/zh-cn/datagrip/download/?section=windows)
2. 选择系统、安装程序格式( 一般是 windows 的 .exe 可执行文件 )
   ![下载安装器](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/下载/DownloadDataGripExe.png "下载安装器" )


### 2、安装
1. 打开下载好的可执行文件( 安装器 )，一路点`下一步`
2. 选择安装位置  
   ![选择安装位置](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/安装/InstallDirctory.png "选择安装位置" )
3. 等待安装  
   ![安装中](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/安装/Installing.png "安装中" )
4. 安装选项勾选`创建桌面快捷方式`、`更新 PATH 变量`  
   ![安装选项](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/安装/InstallOptions.png "安装选项" )
5. 创建快捷方式，点击`安装`  
   ![创建快捷方式](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/安装/CreateShortcut.png "创建快捷方式" )  
   `C:\ProgramData\Microsoft\Windows\Start Menu\Programs\JetBrains` 下面存放了 JetBrains 系列的快捷方式( 比如 IDEA )  
   ![快捷方式存放位置](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/安装/ShortcutDirctory.png "快捷方式存放位置" )
6. 安装完成  
   ![安装完成](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/安装/FinishInstall.png "安装完成" )


### 3、初始化设置
1. 打开 DataGrip  
   ![点它](/assets/images/技术教程类/DataGrip博客系列/ClickIt.png "点它" )
2. 因为笔者用 IDEA 时弄过[ GitHub 学生认证](https://zhuanlan.zhihu.com/p/1929658519281436416)，所以不需要再激活( 也可以先试用30天、参考各种破解方法 )
3. DataGrip 识别到 IDEA，然后导入IDEA的设置  
   ![是否从IDEA导入设置](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/初始化设置/ImportSettings.png "是否从IDEA导入设置" )
   ![选择需要从IDEA导入的设置](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/初始化设置/ImportIdeaSettings.png "选择需要从IDEA导入的设置" )
   ![导入中](/assets/images/技术教程类/DataGrip博客系列/DataGrip安装/初始化设置/Importing.png "导入中" )
