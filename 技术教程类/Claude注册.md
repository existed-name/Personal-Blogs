# 虚拟手机号+Claude网页版注册  

---
​
### 1、背景：
谷歌账号登录Claude后需要选择国家 + 电话，在列表找了几圈没看到+86的电话，一问 Grok 才知道这玩意儿得用国外电话/虚拟电话来接收验证码。于是按照AI的教程整理了以下注册Claude的操作步骤

### 2、步骤

（1）首先得有**稳定节点**（比如机场订阅，免费节点注册的Google账号容易被封😭） + Google账号

（2）去虚拟手机号码平台购买**虚拟号码**，这里用的是 [GrizzlySMS](https://grizzlysms.com/cn)  
![灰熊SMS主页](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/GrizzlySMS.png)

①用自己的邮箱注册（看他发过来的邮件才知道这是个俄罗斯网站） → 进入网站主页 → 搜索服务 Claude  
![选择服务](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/SelectService.png)  
![搜索Claude服务](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/SearchForClaude.png)

②点击 Claude → 选择接收验证码的号码对应的国家（根据个人的VPN来选，比如我用的美国节点，就选美国）
![选择对应的国家](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/SelectCountry.png)

③冲钱（好像是1个号码很便宜，一次性冲进去可以避免频繁交易、买多个号码用） → 这里选的支付宝
![选择支付方式](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/Payment-Alipay.png)  
![支付宝](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/Alipay-TopUp.png)  
![选择套餐](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/PayForPlan.png)  
🥹


④复制卡密（卡号） → 粘贴到步骤③的卡密输入框里面
![复制卡号](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/CopyCardNumber.png)

⑤重复①②选择Claude服务 → 现在就需要验证码了
![等待验证码](assets/images/技术教程类/Claude注册/虚拟手机号平台界面/WaitingForSMS.png)

（3）然后打开Claude

①为了比较干净地注册，我把Edge的浏览数据清空了，然后开的无痕浏览
![Claude主页](assets/images/技术教程类/Claude注册/Claude界面/Claude.png)

②`Continue with Google`（需要用绑定Google账号的手机进行短信验证，Google验证码可能要多等一会儿才到）
![用Google账号登录](assets/images/技术教程类/Claude注册/Claude界面/Claude-ContinueWithGoogle.png)

③把刚刚的虚拟手机号复制过来，发送验证码 → 回到虚拟手机号平台，接收验证码 → 复制粘贴过来  

④成功注册，然后一系列基本设置  
那肯定得Free了😋
![选择套餐](assets/images/技术教程类/Claude注册/Claude界面/Claude-FreePlan.png)  
![初始化设置](assets/images/技术教程类/Claude注册/Claude界面/Claude-Settings.png)  
`Help improve Claude`：允许Anthropic使用对话数据来训练AI

……

⑤别人用Claude写代码，我用Claude写论文报告😇  
![拿下](assets/images/技术教程类/Claude注册/Claude界面/Claude-EnjoyIt.png)


### 3、附

（1）用Google登录Claude时，有网络波动可能会登陆失败，需要检查网络（比如重启Clash）重新登录

（2）部分注意事项 ①没收到验证码可以刷新虚拟手机号平台、换新号码 ②Claude验证码报错可能需要换节点、换号码、刷新浏览器缓存重试  
![注意事项](assets/images/技术教程类/Claude注册/附/Attention.png)

（3）好像第一次用手机号注册，之后就可以不用手机号、直接Google登录了  
![手机号验证](assets/images/技术教程类/Claude注册/附/OnlyOneUseOfPhoneNumber.png)
