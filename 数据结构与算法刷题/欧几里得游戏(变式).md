1. 题目大意  
A、B 比赛，轮流写 1 个正整数，写的这个数必须是已经写的 2 个数之差，并且不同于已有的数  
现在给出 2 个不相等的正整数，从 A 先开始，判断谁会赢 

2. 最容易想到找规律
   1. 输入3 1
      * 第1轮，A 3-1=2 => 3、1、2，B 无解
   2. 输入5 2
      * 第1轮：A 5-2=3 => 5、2、3，B 3-2=1 => 1、2、3、5  
      * 第2轮：A 5-1=4 => 1、2、3、4、5，B 无解  
      * **猜想①**出现[1, 最大那个数]的所有数时，游戏结束，写了(最大数 - 2)个数上去，所以只需要判断最大那个数的奇偶性，奇数A赢，偶数B赢
   3. 输入4 2
      * 第1轮：A无解
   4. 输入9 3
      * 第1轮：A 9-3=6 => 3、6、9，B 无解
      * **猜想②**输入a、b，b=na(a≠1)，那么最终会出现n个数，除开a、b，一共输入了(n-2)个数，n为偶数则B赢，奇数则A赢
   5. 输入9 6
      * 第1轮：A 9-6=3 => 9、6、3，B 无解
   6. 输入20 12
      * 第1轮：A 20-12=8 => 20、12、8，B 12-8=4 => 20、12、8、4
      * 第2轮，A 无解
      * **猜想③**a、b的最大公因数 = 最后出现的数的个数，GCD 为奇数则A赢，偶数则B赢
   7. **总结**
      * **关键看最大公因数**
      * GCD = 1( 互为质数/素数 )会出现 max( a, b )个数 => 需要写( max - 2 )个数上去 => max 为奇数则A赢，偶数则B赢
      * GCD > 1( 不互质 )会出现 GCD 个数 => 需要写( GCD - 2 )个数上去 => GCD 为奇数则A赢，偶数则B赢

3. 数学推导
    1. 输入 a、b( a > b, 且均为正整数 )
        * 设 gcd = GCD( a, b )
        * 则 a = m * gcd, b = n * gcd( m > n, 且互为质数 )
    2. 作差 
       * a - b = gcd( m - n )
       * b - ( a - b ) = 2b - a = gcd( 2n - m )
       * 那么之后不管怎么作差，得到的结果都是 gcd 的倍数
       * 于是我们把所有的 gcd 约掉，只看 m、n
    3. 化简
        * 现在问题变成了输入互质的 m、n，判断谁先赢
        * 既然 m、n 互质，由找规律发现的结论，一共会出现[ 1, max( m, n ) ] = [ 1, m ]个数，也就是会写上去 m - 2 个数
        * 所以 m 为奇数时，A可以写完数字从而获胜，m 为偶数则B赢
    4. **总结**
        * 只需要看 max( a, b ) / gcd( a, b ) 的奇偶性即可
        * 结果为奇数 A 获胜，偶数 B 赢
