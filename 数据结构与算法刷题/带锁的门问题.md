1. 题目大意  
   有N个关闭的门，从这些门前经过N次，第i次经过的时候把【每i个】门的开关状态切换一次，比如第3次，切换3、6、9...的开关状态。求经过N次后打开的门有多少


2. 相对容易想到的方法  
   模拟N个门，循环遍历修改开关状态，最后统计打开的门
    ``` java
    public static List< Boolean > doorStates = 全部初始化为关闭状态( false );
    public static void loop( int n ){
        for ( int ithPass = 1; ithPass <= n; ++ithPass ){ // 模拟第 i 次经过
            for ( int idx = ithPass - 1; // 从第 i 个门开始，对应的下标为 i - 1
                    idx < n; idx += ithPass // 每 i 个门就要切换开关状态
                ){
                boolean state = doorStates.get( idx );
                doorStates.set( idx, !state );
            }
        }
    }
    ```

3. 数学推导
   1. 门的开关状态取决于开关次数的奇偶性 => 最开始关闭，则开关奇数次为打开，偶数次保持原样
   2. 门的开关次数：第 k 个门，如果 k1、k2 是 k 的约数，那么第 k 个门在第 k1、k2 次经过的时候也被开关过。所以开关次数取决于约数的个数
    * **数学知识①**约数--能够整除某个数的数，比如6的约数是1、2、3、6
   3. 于是有奇数个约数，门的开关次数就为奇数，门最后就是打开的
    * 于是需要判断**哪些门的约数是奇数个**
    * **数学知识②**完全平方数（1、4、9、16...）的约数有奇数个，非完全平方数的约数有偶数个
   4. 于是要看最后有多少门打开 => 有多少门的约数是奇数个 => **N 以内有多少个完全平方数**
    * **数学知识③**对 N 开平方，向 0 取整，即可得到 N 以内完全平方数的个数
    * 比如 10 以内有 1、4、9 → 根号10 ≈ 3
