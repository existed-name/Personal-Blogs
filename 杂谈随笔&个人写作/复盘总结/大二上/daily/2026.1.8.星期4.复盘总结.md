## （二）继续学SQL
### 1、对表格结构的操作
#### （1）增
```
# 添加字段
`alter table 表名 add 添加的字段名 字段类型 [约束] [comment注释];`

# 理解记忆：修改了表格的组成部分（结构） => 需要“改变”表格 => 用 `alter table` 开头
# 这些指令都会先在开头标记“大方向要干什么”（`alter/create/drop table`），之后再补充细节
# 约束：可不可以为空 `null/not null`、`default`默认值、`unique`、`primary key`
# 中括号 [] 包裹的内容可加可不加
# 大部分情况，[约束]  [comment] 的顺序可换

# 创建表
create table [if not exists] 表名 (
	各种字段
) [comment注释] ;
```

#### （2）删
```
# 删除字段；add/drop = add/drop column（毕竟确实增删字段看起来就是增删了1列）
alter table 表名 drop 要删的字段名;

# 删除表
drop table [if exists] 要删除的表名;
```

#### （3）改
```
# modify不改名，只“修改”字段的属性（类型、约束、注释）
`alter table 表名 modify 字段名 新的字段类型 [约束] [comment注释];`

# change“改变”整个字段（名称、类型）
`alter table 表名 change 字段名 新的字段名（也可以跟原来一样） 新的字段类型 [约束] [comment注释];`

# 重命名
`rename table 原来的表名 to 新的表名;`  =  alter table 原来的表名 rename to 新的表名
```

#### （4）查
```
# 展示当前数据库文件夹（`select database();`）的所有表格
`show tables;`

# 描述这个表 => 从它的组成结构来描述 => 展示表的结构
`desc 表名;`  =  describe 表名;`  ≈  `show [full] columns from 表名;`

# 查看创建表的源代码/SQL语句
`show create table 表名;`
```


### 2、对表格数据的操作
#### （1）增
```
# 添加1行数据
insert into 表名 ( 字段1的名字, 字段2的名字... ) values( 字段1的值, 字段2的值... );
insert into 表名 values( 所有字段的值 );
# ①类似调用构造器创建对象
# ②如果字段有默认值，可以不用添加这个字段的数据
# ③所有字段都没有默认值、需要手动初始化时，有点像全参构造器

# 添加多行数据
insert into 表名 ( 字段1的名字, 字段2的名字... ) values( 字段1的值, 字段2的值... ), ( 字段1的值, 字段2的值... )...;
```

#### （2）删
```
# 删除指定行
delete from 表名 [ where 条件 ];

# 删除 id = 1 的那行
delete from user_table where id = 1; 

# 删除所有行
delete from 表名;
```

#### （3）改
```
# 修改指定的行（**不加 [where 条件] 则修改所有行**）
update 表名 set 字段1的名字 = 数据, 字段2的名字 = 数据...[where 条件];
update user_table set username = '老王', age = 66 where id = 1;
```
